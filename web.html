Budget Calculator

This calculator was requested by EECU. The app allows users to input their career salary and expenses to calculate their monthly budget. For this project, we had to rotate around the classroom to help our classmates develop our app. So, I didn’t create this website all by myself. I had many fellow talented classmates help me out with coding the layout of the calculator because this wasn’t easy.

Text Adventure

This was a very fun project to work on, and I wish I had more time to work on it. Basically, in class I created a text adventure website that allows the user to be in the story by clicking on choices themselves. This was also where my CSS skills shined, as I was able to achieve the design I envisioned in my mind beforehand. Therefore, I am still proud of this work.
